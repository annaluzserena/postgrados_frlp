# 🎨 Wireframes & Prototipos - Sistema Académico

Este repositorio contiene los **wireframes**, **mockups** y **prototipos** del proyecto de Sistema Académico Modular.

El objetivo es centralizar el diseño visual y funcional de cada módulo antes de su implementación, facilitando la comunicación entre equipos, validaciones y planificación del desarrollo.

---

## 🧩 Estructura de Módulos

| Módulo | Descripción | Estado |
|--------|-------------|--------|
| **CORE** *(todos los equipos)* | Inscripción, legajo digital, workflow de estados y gestión documental | 🟡 En kickoff |
| **MOD-B** | Gestión Docente: asistencia digital, calificaciones y recordatorios automáticos | 🔴 Pendiente |
| **MOD-C** | Seguimiento de Graduación: TFI/Tesis y semáforo de riesgo académico | 🔴 Pendiente |
| **MOD-D** | Analytics & Reporting: dashboards, estadísticas por cohorte y exportación | 🔴 Pendiente |

---

## 📌 Contenido del Repositorio

Dentro de este repositorio se almacenarán:

- 📐 Wireframes
- 🎨 Mockups UI/UX
- 🖥️ Prototipos navegables
- 📄 Flujos de pantallas
- 🧠 Ideas visuales y estructuras funcionales

---

## 🎯 Objetivo

Diseñar y validar la experiencia visual y funcional del sistema antes del desarrollo técnico de cada módulo.
